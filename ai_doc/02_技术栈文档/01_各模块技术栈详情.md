# TestHub 各模块技术栈详情

---

## 1. 总体技术栈

### 后端技术栈

| 类别 | 技术 | 版本/说明 |
|------|------|-----------|
| **Web 框架** | Django | 4.2 |
| **API 框架** | Django REST Framework (DRF) | 含 drf-spectacular（Swagger/ReDoc 文档） |
| **认证** | djangorestframework-simplejwt | JWT（access 60min + refresh 7天 + 轮换黑名单） |
| **Token 过期管理** | django-rest-framework-simplejwt.token_blacklist | 刷新后旧 token 加入黑名单 |
| **CORS** | django-cors-headers | 跨域支持（含跨域 Cookie） |
| **数据库** | MySQL 8.0+ | utf8mb4 字符集 |
| **ORM** | Django ORM | 原生 QuerySet |
| **异步任务** | Celery | Redis/RabbitMQ 作为 broker |
| **WebSocket** | Django Channels | Redis Channels Layer |
| **过滤/搜索** | django-filter | 集成 DRF 的 FilterBackend |
| **API 文档** | drf-spectacular | Swagger + ReDoc 自动文档 |
| **管理后台** | Django SimpleUI | 美化 Django Admin |
| **缓存** | Django LocMemCache | 本地内存缓存（默认 300s） |
| **HTTP 客户端** | httpx | 接口测试中发送 HTTP 请求 |
| **浏览器自动化** | Selenium | UI 自动化基础引擎 |
| **浏览器自动化** | Playwright | UI 自动化备选引擎 |
| **APP 自动化** | Airtest | Android APP 图像识别自动化 |
| **AI Agent** | browser-use | LangChain 生态的浏览器智能代理 |
| **AI LLM** | LangChain + OpenAI SDK | 兼容多 AI 提供商 |
| **文档处理** | python-docx, PyPDF2 | Word/PDF 文本提取 |
| **Excel 处理** | openpyxl, xlsxwriter | Excel 导入导出 |
| **图片处理** | Pillow, qrcode, python-barcode | 二维码/条形码生成、图片处理 |
| **加密** | pycryptodome | AES 加解密 |

### 前端技术栈

| 类别 | 技术 | 版本/说明 |
|------|------|-----------|
| **框架** | Vue 3 | Composition API |
| **构建工具** | Vite | 快速开发/构建 |

## 2. 各模块技术栈详细分解

### 2.1 用户认证模块

| 层 | 技术 | 用途 |
|----|------|------|
| 后端模型 | Django AbstractUser | 自定义 User 扩展 |
| 后端认证 | djangorestframework-simplejwt | JWT 双 Token 机制 |
| 后端鉴权 | DRF IsAuthenticated | 默认所有 API 需认证 |
| 后端中间件 | 自定义 DisableCSRFMiddleware | CSRF 豁免 |
| 前端 Store | Pinia useUserStore | Token/用户状态管理 |
| 前端路由 | Vue Router beforeEach | 路由守卫（requiresAuth/requiresGuest） |
| 前端 HTTP | Axios 拦截器 | 自动注入 Token + 401 刷新 |

### 2.2 项目管理模块

| 层 | 技术 | 用途 |
|----|------|------|
| 后端模型 | Django Model (FK/M2M) | Project + 成员管理 |
| 后端权限 | DRF 自定义 Permission | 基于项目成员的数据隔离 |
| 后端过滤 | django-filter | 项目名称搜索、状态筛选 |
| 前端组件 | Element Plus Table/Form/Dialog | 列表/表单/对话框 |
| 前端筛选 | Element Plus Select/Input | 下拉筛选、搜索 |

### 2.3 测试用例管理模块

| 层 | 技术 | 用途 |
|----|------|------|
| 后端模型 | Django Model (TestCase + Step + Attachment + Comment) | 用例四表体系 |
| 后端序列化 | DRF ModelSerializer (嵌套) | 含步骤/附件/评论的嵌套序列化 |
| 后端导出 | openpyxl | Excel 导出 |
| 后端导入 | openpyxl | Excel 解析导入 |
| 后端上传 | Django FileField | 附件文件管理 |
| 前端编辑器 | 自定义 Markdown 编辑 | 描述字段 |

| 前端排序 | 拖拽排序（vuedraggable 或原生） | 步骤排序 |
| 前端批量操作 | Element Plus Table selection | 批量删除/导出 |

## 3. 基础设施技术栈

| 类别 | 技术 | 说明 |
|------|------|------|
| **数据库** | MySQL 8.0+ | utf8mb4，通过环境变量配置 |
| **缓存** | Redis | Celery broker + Channels Layer |
| **消息队列** | Celery (Redis/RabbitMQ) | 异步任务调度 |
| **Web 服务器** | Django runserver（开发）/ Gunicorn（生产） | WSGI 服务器 |
| **ASGI** | Daphne / Uvicorn | Channels WebSocket 支持 |
| **文件存储** | 本地文件系统 | media/ 目录 |
| **日志** | Python logging | 文件输出到 logs/app.log |
| **语言/时区** | zh-hans / Asia/Shanghai | 中文本地化 |
