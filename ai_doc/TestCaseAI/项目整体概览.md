## 一、项目架构总览

这是一个 **测试领域的技术项目集合（Monorepo）**，包含 8 个子项目，涵盖：

| 子项目 | 定位 | 技术栈 |

| **snow-platform** | AI 驱动的测试用例生成平台（核心项目） | FastAPI + Vue 3 + MySQL + ChromaDB |
| **snow-testcase** | 测试用例生成实验 / AI 能力原型 | Python + LangChain + Playwright |
| **auto-test-framework** | 企业级自动化测试框架（含 UI/API/Client） | Python + Selenium + Pytest |
| **autoInterfacetest** | Java 接口自动化测试框架 | Java + TestNG + Maven |
| **chat_ai_test** | AI 测试用例协作助手（多智能体） | FastAPI + React + AutoGen |
| **lingma_ai_project** | AutoGen 多代理聊天应用 | FastAPI + Vue 3 + AutoGen |
| **interface-automation** | 接口自动化测试学习项目 | Python + Pytest |
| **源码** | 历史源码备份 | 压缩包归档 |

## 二、各子项目概述

### 2.1 snow-platform（核心项目）

**定位：** 承载完整业务流程的 AI 测试用例生成平台。  
**阶段：** 核心功能已全部完成，可正常使用。  
**核心链路：** 项目管理 → 文档上传 → AI 提取测试点 → AI 生成测试用例 → Excel 导出

**主要功能：**
- 多格式文档上传与解析（PDF/DOCX/MD/YAML/CSV）
- AI 自动提取 PRD 测试点（LiteLLM 调用大模型）
- AI 自动生成测试用例（含步骤和预期结果）
- RAG 知识库管理（ChromaDB 向量检索增强生成质量）
- 测试点/用例手动编辑与校验
- 测试用例导出 Excel
- 飞书 Webhook 通知
- 后台异步生成 + 批次任务追踪
- 系统设置持久化（API Key/模型/Webhook 热更新）

**技术栈：**
- 后端：FastAPI + SQLAlchemy 2.0 async + MySQL 8.0 + LiteLLM + ChromaDB
- 前端：Vue 3 + TypeScript + Element Plus + Vite
- 部署：单进程模式，FastAPI BackgroundTasks 处理异步任务

### 2.2 snow-testcase（AI 测试用例实验项目）

**定位：** 探索 AI 生成测试用例的技术验证项目。  
**阶段：** 实验性项目，包含多个原型 Demo。

**包含模块：**
- **ai-testcase/**：基于 LangChain 的 AI 测试用例生成实验
  - `langchain-demo.py`、`langchain-rag-demo.py` — LangChain 集成 Demo
  - `robot1.py` ~ `robot5.py` — 多机器人/Agent 协作实验
  - `dataAnalysisAgent.py`、`documentGenerateAgent.py` — 数据分析与文档生成 Agent
  - `multitranslateAgent.py`、`codeReviewSystem.py` — 翻译 Agent 和代码审查系统
  - `intelligentCustomer.py` — 智能客服 Demo
- **doc-based-test-generator/**：基于文档的测试用例生成技能包
  - Claude Code 技能包，包含功能/API/性能/自动化测试用例标准参考
  - 支持 Playwright MCP 交互式操作
  - 提供快速开始、使用指南、保存机制等文档
- **features/**：BDD 特性文件（待完善）
- **Playwright MCP 集成配置**：支持浏览器自动化录制

### 2.3 auto-test-framework（企业级自动化测试框架）

**定位：** 成熟的 Python 自动化测试框架，支持 Web/HTTP/Client 三种测试类型。  
**阶段：** 工程化较完善，但被测系统项目名称为占位符。

**框架结构：**
- **Base/**：基础组件层
  - `baseAutoWeb.py` — Web 自动化基类（Selenium 封装）
  - `baseAutoHttp.py` — HTTP 接口自动化基类（requests 封装）
  - `baseAutoClient.py` — Client 自动化基类
  - `baseUtils.py` — 工具函数
  - `baseData.py` — 数据驱动
  - `baseExcel.py` — Excel 数据处理
  - `baseYaml.py` — YAML 配置文件处理
  - `baseLogger.py` — 日志封装
  - `basePath.py` — 路径管理
  - `baseContainer.py` — 容器管理
  - `baseGuiRun.py` — GUI 运行器
  - `baseSendEmail.py` — 邮件发送
- **PageObject/**：页面对象层
  - `p01_client_xsglxt/` — 销售管理系统客户端测试
  - `p02_web_gjxt/` — Web 公文系统测试（登录/文章/文件管理）
  - `p03_http_gjxt/` — HTTP 接口公文系统测试（登录/文章/文件管理）
  - `project02_auto_test/` — 另一个被测系统
- **Config/**：配置文件（.ini 格式）
- **Data/**：测试数据（YAML/Excel/文本）
- **TestSuits/**：测试用例集合
- **Driver/**：浏览器驱动
- **ExtTools/**：扩展工具（Shell 操作、数据库操作）
- **RunMain/**：运行入口（支持命令行和 GUI）
- **Reports/**：测试报告

**支持特性：**
- 三种自动化类型：Web(selenium)、HTTP(requests)、Client(socket)
- 数据驱动：YAML + Excel 数据源
- 关键字驱动：Page Object 模式
- 日志系统：loguru 级别控制
- 报告系统：HTML 报告自动生成
- 邮件发送：测试结果邮件通知
- 数据库操作：MySQL 查询支持
- Shell 操作：远程服务器命令执行

### 2.5 chat_ai_test（AI 测试用例协作助手）

**定位：** 基于多智能体（Multi-Agent）协作的测试用例生成助手。  
**阶段：** 前后端分离，基本功能可用。

**核心特性：**
- **多智能体团队协作**：
  - 测试用例设计师（primary）：根据需求生成测试用例
  - 测试用例评审师（critic）：系统性评审，指出改进建议
  - 用户评审（user）：决定接受或重新设计
- **SSE 流式输出**：实时显示每个智能体的输出
- AutoGen 0.7.5 的 RoundRobinGroupChat 实现
- 文件上传提取文本功能
- React 前端 + Gemini/智谱风格 UI

**后端 API：**
- `POST /api/chat/stream` — SSE 流式聊天
- `POST /api/chat/message` — 非流式聊天
- `POST /api/upload` — 文件上传

### 2.6 lingma_ai_project（灵码 AI 项目）

**定位：** AutoGen 多代理聊天应用（DeepSeek 模型）。  
**阶段：** 前后端分离，基本功能可用。

**核心特性：**
- DeepSeek 模型集成（OpenAI-compatible API）
- SSE 实时流式对话
- AutoGen RoundRobinGroupChat 多代理协作
- Vue 3 + ant-design-vue 前端
- Gemini 风格简约 UI
- 会话管理（session_id 关联请求和响应）
- 自动重连机制

**后端 API：**
- `POST /api/session` — 创建新会话
- `POST /api/chat` — 发送消息
- `GET /api/stream/{session_id}` — SSE 流式响应
