# V1.0 基础平台 - 开发提示词

## 需求提点

实现 TestPlate 基础平台，包含：用户认证系统（注册/登录/JWT Token/Token刷新）、项目管理（CRUD/成员/权限）、首页门户（导航卡片/Layout布局/国际化）。

关键是搭建稳固的认证体系和项目权限骨架，所有后续版本都依赖此版本。

---

## 执行计划

### 第一阶段：后端认证体系

**步骤1：配置 Django 项目基础设置**
- 配置 MySQL 数据库连接（settings.py）
- 配置 INSTALLED_APPS（rest_framework, simplejwt, corsheaders, users 等）
- 配置 REST Framework（JWT 认证链、全局权限 IsAuthenticated、分页）
- 配置 CORS（允许跨域携带 Cookie）
- 配置 SIMPLE_JWT（access 60min, refresh 7天, 轮换黑名单）

**步骤2：实现自定义 User 模型**
- 创建 users app，定义 User 模型继承 AbstractUser
- 扩展字段：department、position
- 创建 migrations 并迁移

**步骤3：实现认证 API**
- 注册接口 POST `/api/auth/test-register/`
- 登录接口 POST `/api/auth/login/`（返回 access + refresh + user）
- 退出接口 POST `/api/auth/logout/`（refresh token 加入黑名单）
- Token 刷新接口 POST `/api/auth/token/refresh/`
- 用户信息接口 GET/PUT `/api/auth/profile/`
- 用户列表接口 GET `/api/users/`

**步骤4：创建超级用户**
- 执行 createsuperuser 命令
- 配置 SimpleUI 美化 Admin

### 第二阶段：后端项目体系

**步骤5：实现 Project 模型和 API**
- 创建 projects app
- Project 模型：name, description, status, start_date, end_date, members(M2M), created_by
- CRUD ViewSet + Serializer
- 自定义权限：用户只能看到自己参与的项目
- 成员管理接口（添加/移除）

### 第三阶段：前端基础设施

**步骤6：初始化 Vue 3 项目**
- Vite + Vue 3 + Element Plus + Pinia + Vue Router + Axios + vue-i18n
- 配置 Vite 代理到 Django 后端
- 引入 Element Plus 完整组件和图标

**步骤7：实现 Axios 封装和 Token 管理**
- Axios 实例（baseURL: /api）
- 请求拦截器：自动注入 Bearer Token
- 响应拦截器：401 自动刷新 Token，并发请求排队
- Token 预刷新机制（过期前 5 分钟）

**步骤8：实现 Pinia useUserStore**
- 状态：user, accessToken, refreshToken, tokenExpiresAt
- 操作：login, register, logout, refreshAccessToken, fetchProfile, initAuth
- 自动刷新定时器（每 2 分钟检查）

**步骤9：实现路由和守卫**
- 路由配置：/, /login, /register, /home, /profile
- beforeEach：有 token 无 user 时 initAuth
- requiresAuth：未登录跳转 /login
- requiresGuest：已登录跳转 /home

**步骤10：实现登录/注册页面**
- Login.vue：左侧品牌展示 + 右侧登录表单
- Register.vue：注册表单（用户名/邮箱/姓名/密码/确认密码）

**步骤11：实现 Layout 布局**
- 侧边栏（240px）：6 大模块菜单（先用占位符）
- 顶栏（60px）：面包屑 + 语言切换 + 用户下拉
- 主内容区：router-view
- 退出登录功能

**步骤12：实现首页门户**
- Home.vue：8 张功能导航卡片
- 新标签页打开

**步骤13：实现国际化框架**
- vue-i18n 配置（zh-cn 默认, en）
- 18 个语言文件模块的基础翻译
- Element Plus 国际化集成
- 语言切换组件

**步骤14：实现项目管理前端**
- ProjectList.vue：搜索/状态筛选/CRUD
- ProjectDetail.vue

**步骤15：实现个人资料页**
- UserProfile.vue：基本信息 + 修改密码

## 实现流程

### 后端流程
```
1. Django settings 配置 → 数据库/DRF/JWT/CORS
2. users.User 模型 → migrations
3. users.views + serializers + urls
4. projects.Project 模型 → migrations
5. projects.views + serializers + urls
6. 测试 API → createsuperuser
```

### 前端流程
```
1. Vite 项目初始化 → 安装依赖
2. Axios 封装 → Pinia useUserStore → Router 守卫
3. Login.vue → Register.vue
4. Layout.vue（侧边栏+顶栏+主内容）
5. Home.vue（导航卡片）
6. vue-i18n 配置 → 语言文件
7. ProjectList.vue → ProjectDetail.vue
8. UserProfile.vue
```
