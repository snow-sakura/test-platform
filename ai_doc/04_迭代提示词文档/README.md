# 迭代版本提示词 - 总览

## 提示词使用说明

每个迭代版本的提示词文档包含以下核心内容：

1. **需求提点**：从迭代需求文档中提炼的核心需求要点，作为 AI 编码的输入
2. **执行计划**：将需求分解为具体的开发任务步骤，明确先后顺序
3. **实现流程**：前后端分别的实现流程指引
4. **验证清单**：用于确认该版本开发完成的检查项

## 如何使用这些提示词

将提示词文档中的「需求提点」和「执行计划」部分作为 prompt 输入给 AI 编码助手，AI 将按照计划逐步实现该版本的功能。
