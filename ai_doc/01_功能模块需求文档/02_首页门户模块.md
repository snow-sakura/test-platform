# 02 - 首页门户模块

## 模块定位
平台入口页面，提供各功能模块的导航入口，展示 TestHub 品牌形象和核心能力。

---

## 后端需求

### 2.1 无独立后端 API
首页门户为纯前端导航页，不依赖独立的后端 API。但需确保用户认证状态有效以展示页面。

### 2.2 间接依赖
- 依赖用户认证模块：需已登录才能访问
- 各模块的仪表盘数据通过各自 API 独立加载

---

## 前端需求

### 2.3 页面组件

| 组件 | 路由 | 功能 |
|------|------|------|
| Home.vue | `/home` | 门户仪表盘，展示 8 张功能导航卡片 |

### 2.4 功能卡片导航（8 大模块入口）

| 卡片 | 跳转路由 | 说明 |
|------|----------|------|
| AI 用例生成 | `/ai-generation` | 需求分析与 AI 生成测试用例 |
| 接口测试 | `/api-testing` | HTTP/WebSocket 接口测试 |
| UI 自动化测试 | `/ui-automation` | Selenium/Playwright UI 自动化 |
| 数据工厂 | `/data-factory` | 80+ 测试数据生成工具 |
| APP 自动化测试 | `/app-automation` | Android APP 自动化测试 |
| AI 智能模式 | `/ai-intelligent-mode` | Browser-Use AI 驱动测试 |
| AI 评测师 | `/ai-generation/assistant` | Dify AI 对话助手 |
| 配置中心 | `/configuration` | AI 模型/提示词/环境配置 |

### 2.5 全局功能
- **语言切换**：右上角下拉，支持中文/英文，使用 vue-i18n
- **用户下拉**：个人资料跳转、退出登录
- **新标签页打开**：点击卡片在新标签页打开对应模块

### 2.6 国际化
- **语言包**：`zh-cn`（默认）和 `en` 两种语言
- **18 个语言文件模块**：api-testing.js、auth.js、common.js、configuration.js、data-factory.js、execution.js、nav.js、notification.js、project.js、report.js、requirement.js、review.js、testcase.js、ui-automation.js、version.js、assistant.js
- **Element Plus 国际化**：通过 `<el-config-provider>` 动态切换

---

## 布局组件

### 2.7 Layout（/layout/index.vue）
- **侧边栏**（240px）：根据当前模块动态渲染菜单项，支持 6 个模块菜单
- **顶栏**（60px）：面包屑导航 + 语言切换 + 用户下拉
- **主内容区**：`<router-view>` 渲染子路由

### 2.8 6 大模块菜单
1. **AI 用例生成**：需求分析、用例生成、AI模型配置、提示词配置
2. **接口测试**：仪表盘、项目管理、接口管理、自动化、历史、环境、定时任务、报告、通知
3. **UI 自动化**：仪表盘、项目、元素、用例、脚本、套件、执行、报告、定时任务、通知
4. **APP 自动化**：仪表盘、项目、设备、元素、用例、套件、执行、报告、定时任务、通知
5. **AI 智能模式**：AI 测试、AI 用例、AI 执行记录
6. **配置中心**：AI 模型、提示词、生成配置、UI 环境、APP 环境、智能模式、Dify

---

## 数据流向
```
Home.vue → 点击导航卡片 → 新标签页打开对应模块 → Layout渲染侧边栏 + 子路由
                                                      ↓
                                              Axios拦截器自动注入Token
```
