# 04 - 测试用例管理模块

## 模块定位
测试用例的核心管理模块，支持手动创建、编辑、查看测试用例，包含测试步骤、附件和评论管理。是平台最重要的基础数据模块之一。

---

## 后端需求

### 4.1 测试用例模型（TestCase）

| 字段 | 类型 | 说明 |
|------|------|------|
| project | FK(Project) | 关联项目 |
| title | CharField(500) | 用例标题 |
| description | TextField | 用例描述 |
| preconditions | TextField | 前置条件 |
| priority | CharField(20) | 优先级：low/medium/high/critical |
| status | CharField(20) | 状态：draft/active/deprecated |
| type | CharField(20) | 类型：functional/performance/security/compatibility/other |
| author | FK(User) | 创建者 |
| created_at | DateTimeField | 创建时间 |
| updated_at | DateTimeField | 更新时间 |

### 4.2 测试步骤模型（TestCaseStep）

| 字段 | 类型 | 说明 |
|------|------|------|
| testcase | FK(TestCase) | 关联用例 |
| step_number | PositiveIntegerField | 步骤序号 |
| action | TextField | 操作描述 |
| expected_result | TextField | 预期结果 |
| created_at | DateTimeField | 创建时间 |

### 4.3 用例附件模型（TestCaseAttachment）

| 字段 | 类型 | 说明 |
|------|------|------|
| testcase | FK(TestCase) | 关联用例 |
| file | FileField | 附件文件 |
| filename | CharField(255) | 原始文件名 |
| file_size | BigIntegerField | 文件大小 |
| uploaded_by | FK(User) | 上传者 |
| uploaded_at | DateTimeField | 上传时间 |

### 4.4 用例评论模型（TestCaseComment）

| 字段 | 类型 | 说明 |
|------|------|------|
| testcase | FK(TestCase) | 关联用例 |
| author | FK(User) | 评论者 |
| content | TextField | 评论内容 |
| created_at | DateTimeField | 创建时间 |

### 4.5 API 端点

| 端点 | 方法 | 功能 |
|------|------|------|
| `/api/testcases/` | GET | 用例列表（支持项目/优先级/状态/类型筛选、搜索、排序、分页） |
| `/api/testcases/` | POST | 创建用例（含步骤） |
| `/api/testcases/{id}/` | GET | 用例详情（含步骤、附件、评论） |
| `/api/testcases/{id}/` | PUT/PATCH | 更新用例 |
| `/api/testcases/{id}/` | DELETE | 删除用例 |
| `/api/testcases/{id}/steps/` | POST | 添加测试步骤 |
| `/api/testcases/{id}/steps/{step_id}/` | PUT/DELETE | 更新/删除步骤 |
| `/api/testcases/{id}/attachments/` | POST | 上传附件 |
| `/api/testcases/{id}/attachments/{att_id}/` | DELETE | 删除附件 |
| `/api/testcases/{id}/comments/` | GET/POST | 评论列表/发表评论 |
| `/api/testcases/export/` | GET | 导出用例（Excel） |
| `/api/testcases/import/` | POST | 批量导入用例（Excel） |
| `/api/testcases/batch_delete/` | POST | 批量删除 |

### 4.6 核心业务逻辑
- **项目权限过滤**：列表仅返回用户有权限的项目的用例
- **步骤排序**：步骤按 step_number 排序，支持插入和重排
- **导出格式**：Excel 导出含标题、描述、前置条件、优先级、步骤等
- **批量导入**：Excel 解析后批量创建用例和步骤

## 前端需求

### 4.7 页面组件

| 页面 | 路由 | 功能 |
|------|------|------|
| TestCaseList.vue | `/ai-generation/testcases` | 用例列表，支持搜索/筛选/批量操作/导出 |
| TestCaseForm.vue | `/ai-generation/testcases/create` | 新建用例表单 |
| TestCaseDetail.vue | `/ai-generation/testcases/:id` | 用例详情（含步骤、附件、评论） |
| TestCaseEdit.vue | `/ai-generation/testcases/:id/edit` | 编辑用例 |

### 4.8 列表页功能
- **搜索**：按标题模糊搜索
- **筛选**：项目下拉、优先级下拉、状态下拉、类型下拉
- **表格列**：ID、标题、项目、优先级标签、状态标签、类型、创建者、更新时间、操作
- **批量操作**：勾选多条后批量删除、批量导出
- **操作按钮**：查看、编辑、删除、复制

### 4.9 表单页功能
- **基本信息**：标题（必填）、描述（Markdown）、前置条件、优先级选择、类型选择
- **测试步骤**：动态添加/删除步骤行，每行含操作描述和预期结果，支持拖拽排序
- **附件上传**：支持拖拽上传，显示文件列表，可删除

### 4.10 详情页功能
- **Tab 切换**：基本信息、测试步骤、附件、评论
- **评论功能**：发表评论、查看评论列表、删除自己的评论
- **步骤展示**：序号、操作、预期结果，表格展示
- **附件下载**：点击下载附件

---

## 数据流向
```
TestCaseList.vue → GET /api/testcases/?project=&priority=&status= → 后端过滤 → 分页返回
TestCaseForm.vue → POST /api/testcases/ (含steps数组) → 创建用例+步骤 → 返回详情
TestCaseDetail.vue → GET /api/testcases/:id → 返回用例+步骤+附件+评论
Excel导入 → POST /api/testcases/import/ → 解析Excel → 批量创建
```
