# TestPlate 功能模块需求文档 - 总览

## 项目概述

TestPlate 是一个 AI 驱动的测试管理平台，当前采用 **FastAPI 0.136+（后端）+ Next.js 14（前端）** 架构，提供测试用例管理、API 测试、UI 自动化测试、APP 自动化测试、AI 智能测试、需求分析及测试用例生成等能力。

## 功能模块全景

```
TestPlate 测试管理平台
├── 1. 用户认证与权限模块
├── 2. 首页门户模块
├── 3. 项目管理模块
├── 4. 测试用例管理模块
├── 5. 测试套件管理模块
├── 6. 版本管理模块
├── 7. 评审管理模块
├── 8. 执行管理模块
├── 9. 报告管理模块
├── 10. AI用例生成模块（需求分析）
├── 11. AI评测师模块（智能助手）
├── 12. 接口测试模块（API Testing）
├── 13. UI自动化测试模块
├── 14. APP自动化测试模块
├── 15. AI智能模式模块（Browser-Use）
├── 16. 数据工厂模块
├── 17. 配置中心模块
└── 18. 通知管理模块
```
