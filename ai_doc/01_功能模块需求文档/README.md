# TestPlate 功能模块需求文档 - 总览

## 项目概述

TestPlate 是一个 AI 驱动的测试管理平台，采用 Django 4.2（后端）+ Vue 3（前端）架构，提供测试用例管理、API 测试、UI 自动化测试、APP 自动化测试、AI 智能测试、需求分析及测试用例生成等能力。

## 功能模块全景

```
TestPlate 测试管理平台
├── 1. 用户认证与权限模块
├── 2. 首页门户模块
├── 3. 项目管理模块
├── 4. 测试用例管理模块
├── 5. 测试套件管理模块
├── 6. 版本管理模块
├── 7. 评审管理模块
├── 8. 执行管理模块
├── 9. 报告管理模块
├── 10. AI用例生成模块（需求分析）
├── 11. AI评测师模块（智能助手）
├── 12. 接口测试模块（API Testing）
├── 13. UI自动化测试模块
├── 14. APP自动化测试模块
├── 15. AI智能模式模块（Browser-Use）
├── 16. 数据工厂模块
├── 17. 配置中心模块
└── 18. 通知管理模块
```

## 模块关系图

```
用户认证 ──→ 首页门户
    │
    ├──→ 项目管理 ──→ 测试用例管理 ──→ 测试套件管理
    │        │              │
    │        ├──→ 版本管理  │
    │        │              │
    │        ├──→ 评审管理 ──┤
    │        │              │
    │        ├──→ 执行管理 ──┤
    │        │              │
    │        └──→ 报告管理 ──┘
    │
    ├──→ AI用例生成（需求分析→AI生成用例→评审→导入用例库）
    │
    ├──→ AI评测师（Dify AI 对话助手）
    │
    ├──→ 接口测试（项目管理→接口管理→自动化套件→定时任务→报告）
    │
    ├──→ UI自动化测试（元素管理→页面对象→脚本→用例→套件→执行）
    │        └──→ AI智能模式（Browser-Use AI驱动测试）
    │
    ├──→ APP自动化测试（设备管理→元素捕获→场景编排→执行）
    │
    ├──→ 数据工厂（80+测试数据生成工具）
    │
    ├──→ 配置中心（AI模型/提示词/Dify/环境配置）
    │
    └──→ 通知管理（飞书/企微/钉钉 Webhook 通知）
```
