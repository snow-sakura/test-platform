# 13 - UI 自动化测试模块

## 模块定位
基于 Selenium/Playwright 的 Web UI 自动化测试平台。支持元素管理、Page Object 模式、脚本编辑、测试用例/套件管理、执行追踪、截图管理和定时任务。

---

## 后端需求

### 13.1 模型体系（18 个核心模型）

**UiProject**：项目名称、描述、URL、浏览器类型、状态、成员、创建时间

**LocatorStrategy（定位策略）**：策略名称、类型（id/name/css/xpath/class/text/等）、描述、优先级

**Element（UI 元素）**：
| 字段 | 类型 | 说明 |
|------|------|------|
| project | FK(UiProject) | 关联项目 |
| name | CharField(200) | 元素名称 |
| locator_type | CharField(20) | 定位器类型 |
| locator_value | CharField(500) | 定位器值 |
| backup_locators | JSONField | 备份定位器列表 |
| group | FK(ElementGroup) | 元素分组 |
| page_url | URLField | 所在页面 URL |
| created_at / updated_at | DateTimeField | 时间戳 |

**ElementGroup（元素分组）**：名称、项目、父分组（树形）、排序

**PageObject（页面对象）**：
| 字段 | 类型 | 说明 |
|------|------|------|
| project | FK(UiProject) | 关联项目 |
| name | CharField(200) | 页面名称 |
| url | URLField | 页面 URL |
| elements | M2M(Element) | 页面元素（通过 PageObjectElement） |
| generated_code | TextField | 自动生成的 Page Object 代码 |

**PageObjectElement**：page_object、element、alias（别名）、order

**TestScript（测试脚本）**：
| 字段 | 类型 | 说明 |
|------|------|------|
| project | FK(UiProject) | 关联项目 |
| name | CharField(200) | 脚本名称 |
| page_object | FK(PageObject) | 关联页面对象 |
| steps | JSONField（通过 ScriptStep） | 脚本步骤 |

**ScriptStep（脚本步骤）**：script、step_number、action_type（click/input/select/wait/assert等）、element、input_value、expected_result

**ScriptElementUsage（脚本元素使用分析）**：script、element、usage_count、context

**UiTestCase（UI 测试用例）**：
| 字段 | 类型 | 说明 |
|------|------|------|
| project | FK(UiProject) | 关联项目 |
| name | CharField(200) | 用例名称 |
| script | FK(TestScript) | 关联脚本 |
| priority | CharField(20) | 优先级 |
| status | CharField(20) | 状态 |
| test_data | JSONField | 测试数据 |

**UiTestSuite（UI 测试套件）**：名称、项目、testcases（M2M + order）

**TestExecution（测试执行）**：suite/testcase、status（running/completed/failed/aborted）、started_at、completed_at、result

**Screenshot（截图）**：execution、step、image、captured_at

**OperationRecord（操作记录）**：执行ID、步骤、操作类型、详情、时间戳

**UiScheduledTask（定时任务）**：名称、套件、cron、触发方式、状态、上次执行

**UiNotificationLog（通知日志）**：task、notification_config、status、sent_at、retry_count

**UiTaskNotificationSetting**：task、notification_config、is_active

**UiEnvironment（环境配置）**：浏览器类型、窗口大小、超时、截图设置等

### 13.2 API 端点（22 个 ViewSet）

**项目管理**：CRUD

**定位策略**：CRUD + 优先级排序

**元素管理（核心）**：
| 端点 | 方法 | 功能 |
|------|------|------|
| `/api/ui-automation/elements/` | GET/POST | 元素列表/创建 |
| `/api/ui-automation/elements/{id}/` | GET/PUT/DELETE | 详情/更新/删除 |
| `/api/ui-automation/elements/{id}/validate_locator/` | POST | **验证定位器**（在浏览器中实时测试） |
| `/api/ui-automation/elements/{id}/usages/` | GET | 查看元素使用情况 |
| `/api/ui-automation/elements/{id}/tree/` | GET | 元素树结构 |
| `/api/ui-automation/elements/{id}/add_backup_locator/` | POST | 添加备用定位器 |
| `/api/ui-automation/elements/{id}/generate_suggestions/` | POST | **AI 生成定位建议** |

**元素分组**：CRUD + 树形结构

**页面对象管理**：
| 端点 | 方法 | 功能 |
|------|------|------|
| `/api/ui-automation/page-objects/` | GET/POST | 页面对象 CRUD |
| `/api/ui-automation/page-objects/{id}/generate_code/` | POST | **自动生成 Page Object 代码** |
| `/api/ui-automation/page-objects/{id}/add_element/` | POST | 添加元素到页面 |
| `/api/ui-automation/page-objects/{id}/elements/` | GET | 获取页面元素列表 |

**脚本管理**：
| 端点 | 方法 | 功能 |
|------|------|------|
| `/api/ui-automation/scripts/` | GET/POST | 脚本 CRUD |
| `/api/ui-automation/scripts/{id}/steps/` | GET/POST | 步骤管理 |
| `/api/ui-automation/scripts/{id}/steps/batch_create/` | POST | 批量创建步骤 |
| `/api/ui-automation/scripts/element_usages/analyze_script/` | POST | 分析脚本元素使用 |

**UI 用例管理**：CRUD + copy_case + run + batch_run

**UI 套件管理**：
| 端点 | 方法 | 功能 |
|------|------|------|
| `/api/ui-automation/suites/` | GET/POST | 套件 CRUD |
| `/api/ui-automation/suites/{id}/scripts/` | GET | 获取脚本列表 |
| `/api/ui-automation/suites/{id}/add_script/` | POST | 添加脚本 |
| `/api/ui-automation/suites/{id}/remove_script/` | POST | 移除脚本 |
| `/api/ui-automation/suites/{id}/test_cases/` | GET | 获取用例列表 |
| `/api/ui-automation/suites/{id}/add_test_case/` | POST | 添加用例 |
| `/api/ui-automation/suites/{id}/remove_test_case/` | POST | 移除用例 |
| `/api/ui-automation/suites/{id}/run_suite/` | POST | **执行整个套件** |

**执行管理**：CRUD + run + abort

**截图管理**：CRUD + 按执行/步骤筛选

**操作记录**：按执行 ID 筛选

**定时任务**：CRUD + pause/resume/run_now

**通知系统**：通知日志（retry）+ 任务通知设置

**环境配置**：CRUD

**仪表盘**：project_count、test_case_count、suite_test_case_count、total_execution_count

### 13.3 核心业务逻辑
- **元素定位验证**：使用 Selenium/Playwright 在浏览器中实时查找元素，返回是否找到及匹配数量
- **Page Object 代码生成**：根据元素列表自动生成 Page Object 类代码
- **脚本执行**：遍历 ScriptStep，解析定位器类型（id/name/css/xpath），调用 Selenium/Playwright API 执行操作
- **备用定位器**：主定位器失败时自动尝试 backup_locators 列表中的备用方案
- **AI 定位建议**：AI 分析页面 HTML 结构，生成可能的元素定位器

### 13.5 元素管理器增强版功能
- **左侧分组树**：多级分组，拖拽移动，右键操作
- **中间元素列表**：按分组筛选，搜索，分页
- **右侧元素详情/编辑**：
  - 定位器类型选择 + 定位器值输入
  - 备用定位器列表（多条，可拖拽排序）
  - 「验证定位器」按钮 → 浏览器实时查找，返回结果
  - 「AI 生成建议」按钮 → AI 分析页面 HTML，返回可选定位器列表
- **元素使用情况**：展示该元素被哪些脚本引用

### 13.6 脚本编辑器增强版功能
- **左侧步骤列表**：可拖拽排序，点击选中
- **右侧步骤编辑**：
  - 操作类型选择（click/input/select/wait/assert/scroll/hover等）
  - 元素选择器（从 Page Object 或元素库中选择）
  - 输入值编辑
  - 预期结果编辑
- **批量添加步骤**：从页面对象添加多个元素对应的操作
- **代码视图**：切换到代码模式查看/编辑生成的脚本代码
- **运行测试**：单步调试或全量运行

---

## 数据流向
```
元素管理 → ElementManagerEnhanced → 验证定位器 → /elements/:id/validate_locator/ → 浏览器实时查找
                                  → AI 建议 → /elements/:id/generate_suggestions/ → AI 分析HTML

脚本执行 → ScriptEditor → POST /scripts/:id/run/ → 遍历步骤 → Selenium/Playwright API → 记录操作历史
套件执行 → POST /suites/:id/run_suite/ → 遍历用例 → 记录执行结果 → 截图 → 通知
```
