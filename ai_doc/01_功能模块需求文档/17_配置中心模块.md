# 17 - 配置中心模块

## 模块定位
平台统一配置管理入口，集中管理 AI 模型配置、提示词配置、生成行为配置、UI/APP 自动化环境配置、Dify 配置和 AI 智能模式配置。

---

## 后端需求

### 17.1 配置模型
> 各模型已在其他模块详述，此处汇总配置中心管理的配置类型。

| 配置类型 | 所属 App | 模型 | 管理端点 |
|----------|----------|------|----------|
| AI 模型配置 | requirement_analysis | AIModelConfig | `/api/requirement-analysis/model-configs/` |
| 提示词配置 | requirement_analysis | PromptConfig | `/api/requirement-analysis/prompt-configs/` |
| 生成行为配置 | requirement_analysis | GenerationConfig | `/api/requirement-analysis/generation-configs/` |
| UI 自动化环境 | ui_automation | UiEnvironment | `/api/ui-automation/environments/` |
| APP 自动化配置 | app_automation | AppConfig | `/api/app-automation/config/` |
| Dify 配置 | assistant | DifyConfig | `/api/assistant/config/dify/` |
| AI 智能模式配置 | ui_automation | (使用 AIModelConfig) | 使用 browser_use_text/vision 角色模型 |
| 统一通知配置 | core | UnifiedNotificationConfig | `/api/core/notification-configs/` |

### 17.2 AI 模型配置核心字段
- **提供商**：DeepSeek / Qwen（通义千问）/ SiliconFlow（硅基流动）/ OpenAI 兼容
- **角色**：testcase_writer / testcase_reviewer / browser_use_text / browser_use_vision
- **连接参数**：API 地址、密钥、模型名称、温度、最大 Token
- **连接测试**：异步调用 OpenAI 兼容 API，60s 超时
- **启用/禁用**：支持 enable/disable 操作

### 17.3 提示词配置
- **类型**：testcase_writer（用例编写）/ testcase_reviewer（用例评审）
- **默认提示词**：从 `docs/tester.md` 和 `docs/tester_pro.md` 加载
- **编辑**：富文本编辑，支持 Markdown
- **启用/禁用**：支持 enable/disable 操作

### 17.4 配置状态检查
- **端点**：`/api/requirement-analysis/config/status/`
- **检查内容**：
  - writer/reviewer 模型是否已配置且启用
  - writer/reviewer 提示词是否已配置且启用
  - 生成配置是否已配置且启用
- **返回**：结构化 JSON，标记每项为 configured/enabled/disabled

## 前端需求

### 17.5 页面组件

| 页面 | 路由 | 功能 |
|------|------|------|
| ConfigurationCenter.vue | `/configuration` | 配置中心容器（含过渡动画路由视图） |
| AIModelConfig.vue | `/configuration/ai-models` | AI 模型配置（复用于需求分析模块） |
| PromptConfig.vue | `/configuration/prompts` | 提示词配置 |
| GenerationConfigView.vue | `/configuration/generation` | 生成行为配置 |
| UIEnvironmentConfig.vue | `/configuration/ui-environment` | UI 自动化环境配置 |
| AppSettings.vue | `/configuration/app-environment` | APP 自动化环境配置 |
| AIIntelligentModeConfig.vue | `/configuration/ai-intelligent-mode` | AI 智能模式配置 |
| DifyConfig.vue | `/configuration/dify` | Dify AI 平台配置 |

### 17.6 配置中心主界面功能
- **侧边栏导航**：各配置子模块菜单
- **路由切换动画**：淡入淡出过渡
- **子路由容器**：`<router-view>` 渲染

### 17.7 各配置子页面功能

**AI 模型配置**：
- 表格列表：名称、提供商标签、角色标签、模型名、温度、Token、状态、操作
- 新建/编辑对话框：选择提供商、填写 API 参数、测试连接按钮
- 启用/禁用开关

**提示词配置**：
- 表格列表：名称、类型标签、状态、操作
- 编辑页：Markdown 编辑器（Monaco Editor）
- 加载默认按钮：从 tester.md/tester_pro.md 加载

**生成行为配置**：
- 表格列表：名称、测试级别、优先级、生成数量、是否自动评审、操作
- 新建/编辑：下拉选择 + 数字输入
- 激活按钮：单例模式，激活一个自动禁用其他

**UI 自动化环境配置**：
- 浏览器类型选择（Chrome/Firefox/Edge/Safari）
- 窗口大小设置
- 超时设置（元素等待、页面加载、脚本执行）
- 截图设置（全屏/视口/元素级）

**APP 自动化环境配置**：
- ADB 路径配置
- 设备超时设置
- 截图设置

**Dify 配置**：
- API URL 输入
- API Key 输入（脱敏显示）
- 测试连接按钮
- 启用/禁用开关

## 数据流向
```
ConfigurationCenter.vue → 侧边栏导航 → router-view渲染子配置页
AI模型配置 → POST /model-configs/ → 保存 → test_connection → 异步测试API连接
提示词配置 → POST /prompt-configs/ → 保存 → load_defaults → 从文件加载
生成配置 → POST /generation-configs/ → 保存 → enable → 单例激活
```
