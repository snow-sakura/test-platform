# 16 - 数据工厂模块

## 模块定位
测试数据生成工具箱，提供 7 大类 80+ 个数据生成/处理工具。支持单独使用、批量生成、变量函数调用，可被接口测试和 UI 自动化测试通过 `${function_name()}` 语法引用。

---

## 后端需求

### 16.1 模型

**DataFactoryRecord（使用记录）**：
| 字段 | 类型 | 说明 |
|------|------|------|
| user | FK(User) | 用户 |
| tool_name | CharField(100) | 工具名称 |
| tool_category | CharField(20) | 分类 |
| tool_scenario | CharField(20) | 使用场景 |
| input_data | JSONField | 输入参数 |
| output_data | JSONField | 输出结果 |
| is_saved | BooleanField | 是否保存 |
| tags | JSONField | 标签 |
| created_at / updated_at | DateTimeField | 时间戳 |

### 16.2 API 端点

| 端点 | 方法 | 功能 |
|------|------|------|
| `/api/data-factory/` | GET | **历史记录列表**（分页/缓存） |
| `/api/data-factory/` | POST | **执行工具并保存** |
| `/api/data-factory/{id}/` | DELETE | 删除记录 |
| `/api/data-factory/categories/` | GET | **获取所有工具分类及工具列表** |
| `/api/data-factory/tags/` | GET | 获取用户所有标签 |
| `/api/data-factory/batch_generate/` | POST | **批量生成数据** |
| `/api/data-factory/statistics/` | GET | 使用统计（分类/场景/最近工具） |
| `/api/data-factory/download_static_file/` | GET | 下载条形码/二维码图片 |
| `/api/data-factory/variable_functions/` | GET | **获取所有变量函数**（供其他模块引用） |

### 16.3 工具体系（7 大类，80+ 工具）

**测试数据工具（test_data，11 个）**：
中文姓名、手机号、邮箱、地址、身份证、公司名、银行卡号、香港身份证、营业执照号、经纬度、用户档案

**JSON 工具（json，10 个）**：
格式化、校验、转 XML、转 YAML、JSONPath 查询、扁平化、对比、路径列表、CSV 互转、Mock 数据

**字符工具（string，9 个）**：
去空格、字符串替换、转义/反转义、字数统计、文本对比、正则测试、大小写转换、格式化

**编码工具（encoding，14 个）**：
条形码生成、二维码生成/解析、时间戳转换、进制转换、Unicode 转换、ASCII 转换、颜色转换、Base64 编解码、URL 编解码、JWT 解析、图片 Base64 转换

**随机工具（random，11 个）**：
随机整数、浮点、字符串、UUID、MAC 地址、IP 地址、日期、布尔、颜色、密码、序列

**加密工具（encryption，9 个）**：
MD5、SHA1、SHA256、SHA512、哈希比对、AES 加解密、密码强度检测、盐值生成

**Crontab 工具（crontab，4 个）**：
Cron 生成、解析、下次执行时间、验证

**辅助工具**：
- `china_bank_card.py`：Luhn 算法银行卡号生成器（内置数万 BIN 码）
- `unified_credit_code.py`：统一社会信用代码/营业执照号生成器
- `image_tools.py`：图片 Base64 互转、图片调整、格式验证

### 16.4 变量解析器（core/variable_resolver.py）
- **全局单例**：`resolve_variables()` 函数
- **语法**：`${function_name(args)}`
- **应用场景**：接口测试请求参数、UI 自动化输入值
- **支持范围**：所有 7 大分类工具的函数

## 前端需求

### 16.5 页面组件

| 页面 | 路由 | 功能 |
|------|------|------|
| DataFactory.vue | `/data-factory` | 数据工厂主页面 |

### 16.6 主页面功能
- **双视图模式**：
  - **分类视图**：左侧分类导航（7 大分类），右侧工具卡片列表
  - **场景视图**：按使用场景组织工具
- **工具执行**：
  - 点击工具 → 显示参数表单
  - 填写参数 → 点击生成 → 显示结果
  - 支持一键复制结果
- **批量生成**：设置批量数量 → 一次生成多条数据
- **历史记录**：查看历史生成记录，支持按标签筛选、删除
- **变量函数助手**：展示所有 `${function_name()}` 用法，方便复制到接口测试中使用

### 16.7 DataFactorySelector 组件
- **弹窗选择器**：可在接口测试等模块中弹出，选择数据工厂生成的数据
- **两个 Tab**：按标签筛选 / 按记录搜索
- **预览**：查看记录的输入/输出 JSON
- **选择**：选中后注入到当前编辑的请求参数中

---

## 数据流向
```
DataFactory.vue → GET /categories/ → 获取分类+工具列表 → 渲染UI
选择工具 → 填写参数 → POST /data-factory/ → 执行工具 → 返回结果 + 保存记录
批量生成 → POST /batch_generate/ → 多次执行 → 返回结果列表

接口测试中使用 → ${random_phone()} → variable_resolver → 调用数据工厂工具 → 替换为实际值
```
