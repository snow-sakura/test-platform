# 09 - 报告管理模块

## 模块定位
测试报告生成与数据统计中心，提供 Dashboard 概览、执行状态分布、缺陷分析、趋势图、AI 效能分析等多维度的测试数据可视化。

---

## 后端需求

### 9.1 报告模型（TestReport）

| 字段 | 类型 | 说明 |
|------|------|------|
| project | FK(Project) | 关联项目 |
| name | CharField(200) | 报告名称 |
| report_type | CharField(20) | execution/summary/trend |
| execution | OneToOneField(TestRun) | 关联执行（可选） |
| summary | JSONField | 报告摘要 JSON |
| content | JSONField | 报告内容 JSON |
| generated_by | FK(User) | 生成者 |
| created_at | DateTimeField | 创建时间 |

### 9.2 报告模板模型（ReportTemplate）

| 字段 | 类型 | 说明 |
|------|------|------|
| name | CharField(200) | 模板名称 |
| description | TextField | 模板描述 |
| template_config | JSONField | 模板配置 |
| is_default | BooleanField | 是否默认模板 |
| created_by | FK(User) | 创建者 |
| created_at | DateTimeField | 创建时间 |

### 9.3 API 端点

| 端点 | 方法 | 功能 |
|------|------|------|
| `/api/reports/reports/` | GET/POST | 报告列表/创建 |
| `/api/reports/reports/{id}/` | GET/PUT/DELETE | 详情/更新/删除 |
| `/api/reports/reports/dashboard/` | GET | 概览数据（活跃计划数/进度/通过率/缺陷数） |
| `/api/reports/reports/status_distribution/` | GET | 执行状态分布（passed/failed/blocked等） |
| `/api/reports/reports/defect_distribution/` | GET | 缺陷分布（按优先级） |
| `/api/reports/reports/failed_cases_top/` | GET | 失败用例 TOP10 |
| `/api/reports/reports/execution_trend/` | GET | 每日执行趋势（默认7天） |
| `/api/reports/reports/ai_efficiency/` | GET | AI 效能分析（AI生成vs人工/采纳率/需求覆盖率/节省时间） |
| `/api/reports/reports/team_workload/` | GET | 团队工作量统计（执行数/缺陷发现数） |

### 9.4 核心业务逻辑
- **Dashboard 聚合**：跨项目聚合活跃计划数、总用例数、通过率、总缺陷数
- **趋势图数据**：按天聚合执行记录，计算每日通过/失败/阻塞数
- **缺陷分布**：按优先级（critical/high/medium/low）统计缺陷数
- **AI 效能**：统计 AI 生成用例 vs 人工用例的采纳率、覆盖率、节省时间
- **团队工作量**：按执行人聚合执行数和缺陷发现数

## 前端需求

### 9.5 页面组件

| 页面 | 路由 | 功能 |
|------|------|------|
| AiTestReport.vue | `/ai-generation/reports` | 测试报告仪表盘，含筛选和图表 |
| ReportList.vue | `/ai-generation/reports/list` | 报告列表（占位符） |

### 9.6 仪表盘功能
- **顶部筛选**：项目选择、时间范围选择
- **统计卡片**：活跃计划数、总用例数、通过率百分比、失败趋势
- **图表**：
  - 执行状态分布（饼图/柱状图）
  - 缺陷分布（按优先级）
  - 失败用例 TOP10（横向柱状图）
  - 每日执行趋势（折线图）
  - AI 效能分析（对比图）
  - 团队工作量（柱状图）

### 9.7 数据刷新
- **自动刷新**：定时刷新仪表盘数据
- **筛选联动**：项目和时间变更后重新加载所有图表

---

## 数据流向
```
ReportDashboard → GET /api/reports/reports/dashboard/ → 聚合数据 → 渲染图表
趋势图 → GET /api/reports/reports/execution_trend/?days=7 → 日聚合数据 → 折线图
AI效能 → GET /api/reports/reports/ai_efficiency/ → AI相关统计 → 对比展示
```
