# V1.5 - 测试用例与评审体系

## 版本概述

**业务重点**：建立完整的手工测试管理体系，包括测试用例的全生命周期管理（创建 → 评审 → 执行 → 报告）

**前置依赖**：V1.0（用户认证 + 项目管理）

**后续依赖**：V2.0、V2.5 的自动化测试依赖本版本的用例库

## 里程碑验收标准

1. 完整的测试用例生命周期：创建 → 评审 → 加入执行计划 → 执行追踪 → 报告
2. 用例可包含多步骤、附件、评论
3. Excel 导入/导出功能正常
4. 评审流程完整：分配评审人 → 提交意见 → 自动汇总
5. 执行计划自动生成 TestRun，进度实时计算
6. 报告仪表盘展示关键测试指标
