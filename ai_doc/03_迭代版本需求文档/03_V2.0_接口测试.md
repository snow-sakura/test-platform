# V2.0 - 接口测试

## 版本概述

**业务重点**：实现完整的 HTTP/WebSocket 接口测试能力

**前置依赖**：V1.0（用户+项目）、V1.5（用例库概念，接口测试用例可引用基础用例库）

**后续依赖**：V2.5（UI 自动化参考接口测试的定时任务和通知体系）

## 关键技术点

- **HTTP 客户端**：httpx（异步支持）
- **变量解析**：`${function_name()}` → core.variable_resolver
- **定时调度**：Celery Beat + cron 表达式
- **WebSocket**：Django Channels

---

## 里程碑验收标准

1. 支持 HTTP GET/POST/PUT/DELETE/PATCH/HEAD/OPTIONS 全部方法
2. 支持 JSON/FormData/FormUrlencoded 请求体
3. 支持多级集合树结构
4. 环境变量可在 URL/Body/Headers 中正确替换
5. 套件可批量执行并汇总通过/失败数
6. 定时任务按 cron 表达式自动执行
7. 请求历史可回溯查看
8. 通知可推送到飞书/企微/钉钉
