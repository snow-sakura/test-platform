# V1.0 - 基础平台

## 版本概述

**业务重点**：搭建平台骨架，实现核心基础设施

**前置依赖**：无

**后续依赖**：所有后续版本都依赖 V1.0 的用户认证和项目体系

---

## 需求清单

### 1. 用户认证模块

#### 后端
- [ ] 自定义 User 模型扩展（username/email/first_name/last_name/department/position）
- [ ] JWT 双 Token 认证机制（access_token 60min + refresh_token 7天）
- [ ] 用户注册接口（POST `/api/auth/register/`）
- [ ] 用户登录接口（POST `/api/auth/login/`，返回双 token + 用户信息）
- [ ] 退出登录接口（POST `/api/auth/logout/`，refresh_token 加入黑名单）
- [ ] Token 刷新接口（POST `/api/auth/token/refresh/`，旧 token 自动加入黑名单）
- [ ] 用户信息接口（GET/PUT `/api/auth/profile/`）
- [ ] 用户列表接口（GET `/api/users/`）
- [ ] DRF 全局认证/权限配置（JWT → Token → Session 链，默认 IsAuthenticated）
- [ ] CORS 跨域支持（django-cors-headers）
- [ ] Django Admin 配置（SimpleUI 美化）

#### 前端
- [ ] 登录页面（Login.vue）：用户名/邮箱 + 密码，品牌展示
- [ ] 注册页面（Register.vue）：用户名、邮箱、姓名、密码、确认密码
- [ ] Pinia useUserStore：Token/用户状态管理
- [ ] Axios 拦截器：自动注入 Bearer Token + 401 自动刷新
- [ ] 路由守卫：requiresAuth/requiresGuest + beforeEach 自动恢复状态
- [ ] Token 自动刷新机制：过期前 5 分钟提前刷新
- [ ] 个人资料页面（UserProfile.vue）：基本信息 Tab + 修改密码 Tab

### 2. 项目管理模块

#### 后端
- [ ] Project 模型（name/description/status/start_date/end_date/members/created_by）
- [ ] 项目 CRUD 接口（GET/POST/PUT/DELETE）
- [ ] 基于项目成员的数据权限过滤
- [ ] 项目成员管理接口（添加/移除）

#### 前端
- [ ] 项目列表页面（ProjectList.vue）：搜索、状态筛选、CRUD
- [ ] 项目详情页面（ProjectDetail.vue）
- [ ] 新建/编辑项目对话框表单

### 3. 首页门户模块

#### 前端
- [ ] 首页门户（Home.vue）：8 张功能导航卡片
- [ ] Layout 布局组件（侧边栏 + 顶栏 + 主内容区）
- [ ] 侧边栏动态菜单渲染（按模块）
- [ ] 顶栏：面包屑 + 语言切换（中/英）+ 用户下拉（资料/退出）
- [ ] vue-i18n 国际化框架搭建（中/英双语）
- [ ] Element Plus 国际化集成

## 里程碑验收标准

1. 用户可以注册、登录、退出
2. Token 过期自动刷新，用户体验无感
3. 用户只能看到自己参与的项目
4. 未登录用户无法访问任何功能页
5. 首页展示完整的功能导航卡片
6. 中英文切换功能正常
